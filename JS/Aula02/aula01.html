JavaScrip - Introdução

Três comando básicos para interação com o usuário.

Alert

Para quando quero forçar a atenção do usuário para determinada infomação.

Prompt

Fazer uma interação direta, pedindo algo ao usuário

Confirm

Da duas opções ao usuário para interagir: "Cancelar" e "Ok"

Variáveis

Espaço em memória para guardar valores.

Tipos de variáveis: let, const, var e array

  • Let: tudo que varia, valores flexiveis para armazenar 1 dado;
  • Const: valores fixos, "crachado";
  • Var: primeira forma de declarar uma variável no javacript. Já está absoleta, de forma que só vemos em sistemas legados;
  • Array: é uma váriável que armazena vários valores em um conjunto. Esse conjunto recebe um indice começando em 0.
